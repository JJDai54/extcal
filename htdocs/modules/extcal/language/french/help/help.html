Aide : extCal

Description

eXtCal est un module de calendrier d'événements développé exclusivement pour Xoops, donc il y est parfaitement intégré.

Il utilise l'autorisation et la gestion de groupe de Xoops, permettant ainsi une grande souplesse d'utilisation.

Installer / désinstaller

Aucune mesure spéciale n'est nécessaire. Suivez la procédure d'installation standard, envoyez le dossier « extcal » dans le répertoire .../modules. Installez le module par Admin-> Système-> Modules.

Instructions détaillées sur la façon d'installer un module sur le Manuel des opérations de Xoops

Mode d'emploi

Pour configurer ce module, vous devez :

  • i) Configurez vos préférences pour le module (voir « Préférences ») et éventuellement le bloc Partenaire si vous comptez l'utiliser (voir « blocs »)
  • ii) Vérifiez que vous avez donné à vos groupes d'utilisateurs et blocs, les droits d'accès nécessaires pour utiliser ce module. Les permissions de groupe se définissent dans l'Administration à Modules -> Système -> Groupes.

Des instructions détaillées sur la configuration des droits d'accès des groupes d'utilisateurs sont disponibles dans le Manuel des opérations de Xoops
